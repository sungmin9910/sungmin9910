# 🏎️ Autonomous Driving Simulator

이 프로젝트는 농산물 운반 및 물류 자동화를 위한 자율주행 시뮬레이터입니다.

## 🌟 주요 기능
- **3D 환경**: Three.js를 이용한 실시간 3D 렌더링
- **차량 제어**: 키보드를 이용한 수동 제어 및 자율주행 알고리즘 테스트
- **디지털 트윈 연동**: 실시간 차량 상태 데이터 시각화 (예정)

## 🛠️ 기술 스택
- **Core**: HTML5, JavaScript (ES6+)
- **3D Engine**: Three.js
- **Styling**: Vanilla CSS (Modern Premium Design)
